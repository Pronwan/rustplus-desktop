# Использование папки modifications (Modifications Sharing Guide)

Папка `modification` содержит все файлы пользовательских модификаций для приложения Rust+ Desktop. Вы можете легко делиться этой папкой с другими пользователями или переносить её на другие компьютеры.

## Как использовать папку `modification`

1. **Скопируйте папку**: Отправьте папку `modification` целиком другому человеку или перенесите её на новый компьютер.
2. **Поместите в корень проекта**: Скопируйте полученную папку `modification` в корневую директорию проекта `rustplus-desktop` (туда же, где находятся файлы `RustPlusDesk.sln` и папка `RustPlusDesktop`).
3. **Сборка проекта**:
   - Дважды кликните по файлу `build.bat` в корне проекта или выполните команду `dotnet build` в консоли.
   - Проект автоматически подхватит все `.cs` файлы из папки `modification` при компиляции благодаря настроенным правилам в файле проекта `.csproj`.
4. **Запуск**:
   - Используйте файл `run.bat` в корне для запуска скомпилированного приложения.

## Содержимое папки

- `IMod.cs` — общий интерфейс для создания новых модификаций.
- `ModManager.cs` — менеджер, который инициализирует все моды и выводит их в новую вкладку `🛠 Mods` в настройках приложения.
- `SmartSwitchesMod.cs` — модификация "Умные переключатели" (автоматическое выключение по таймеру или переключение через чат).
- `GridCustomizationMod.cs` — модификация для ручной калибровки координатной сетки (X/Y смещение).
- `MainWindow.Modifications.cs` — расширение основного окна для интеграции модов.
